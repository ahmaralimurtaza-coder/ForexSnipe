# 🎯 ForexSnipe

**AI-Powered Forex Market Intelligence App**

ForexSnipe is a professional Flutter Android app that provides
real-time market data for Forex, Stocks, Crypto, Indices,
Commodities and Futures — all in one place.

---

## 📱 Features

- 💱 **Live Prices** — Forex, Stocks, Crypto, Indices, Commodities, Futures
- 📊 **COT Report** — Real CFTC.gov institutional positioning data
- 📅 **Economic Calendar** — High-impact events with impact filters
- 📰 **Market News** — AI sentiment analysis from multiple sources
- 😊 **Sentiment** — Retail trader positioning (contrarian indicator)
- 📈 **Barchart** — Direct COT chart links
- 🤖 **Forex Sniper AI** — AI market analyst chatbot
- 🌐 **Data Sources** — In-app browser for all sources

---

## 🛠️ Tech Stack

- **Framework:** Flutter (Dart)
- **State Management:** Provider
- **Data Sources:**
    - ExchangeRate API (Forex)
    - CoinGecko (Crypto)
    - Yahoo Finance (Stocks, Indices, Futures)
    - Finnhub (News, Calendar)
    - CFTC.gov (COT Data)
    - NewsAPI, Mediastack (News)

## 📦 Installation

### Prerequisites
- Flutter SDK 3.0+
- Android Studio
- Android device or emulator

### Setup

1. Clone the repository
   git clone https://github.com/YOUR_USERNAME/ForexSnipe.git
   cd ForexSnipe

2. Install dependencies
   flutter pub get

3. Add your API keys
   Create lib/services/api_keys.dart with your keys
   See api_keys.example.dart for format

4. Run the app
   flutter run
